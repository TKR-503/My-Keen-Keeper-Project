
# Project Name: 
My-Keen-keeper-app


# Project Description:

My-Keen-Keeper-App is a modern friendship tracking application that helps users stay connected with friends through quick Call, Text, and Video check-ins. The app provides a clean timeline interface and pir chart to manage and view recent interactions easily.

## Technologies is used:

-Next.js
-React.js
-Tailwind CSS
-Javascript

## 3 key features of my project:

1- Timeline Activity Tracking which Users can log Call, Text, and Video 
   check-ins with friends.

2- Filter timeline activities by Call, Text, or Video interactions    
   or using pie chart by dynamic filtering.

3- Responsive Modern UI which can Clean and mobile-friendly interface 
   built with Tailwind CSS.
